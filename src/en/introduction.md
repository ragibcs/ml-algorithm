## How to Add a New Algorithm

Each algorithm is a single `.md` file in the `src/en/` folder. To add a new one:

1. Create a new `.md` file in `src/en/` (e.g., `src/en/linear-regression.md`)
2. Write the article
3. Add a new entry in `src/SUMMARY.md` under the English section
4. Push — GitHub Actions auto-deploys
