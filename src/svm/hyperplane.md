# Hyperplane

Hyperplane হলো একটি Decision Boundary যা বিভিন্ন Class-কে আলাদা করে।

## বিভিন্ন Dimension-এ Hyperplane

| Dimension        | Hyperplane |
| ---------------- | ---------- |
| 2D               | Line       |
| 3D               | Plane      |
| Higher Dimension | Hyperplane |

## Hyperplane Equation

```math
w^Tx + b = 0
```

যেখানে:

* `w` = Weight Vector
* `x` = Feature Vector
* `b` = Bias
